1/24から５泊６日のベトナム旅行に行ってきました。前半はホーチミン、後半はダナンです。自分も妻もパクチー好きだからベトナム料理もきっと美味しく食べられそうなのと、前回の海外旅行で行った[チェコ](/blog/20241112)は遠すぎたから今回は近場にしよう、と思ってベトナムにしました。

## 1日目
ホーチミン1日目。チャイナエアラインで沖縄→台湾→ホーチミンで行きました。乗り継ぎ時間含めて５時間ぐらいで行けました。

{{< figure src="/images/20260201/1/1.jpg" title="ホーチミンのタンソンニャット国際空港" class="center" >}}

まず到着して思ったのが暑い！沖縄は20度ぐらいだけどホーチミンは30度。

ベトナムの移動手段の基本はタクシー。事前にGrabというタクシーアプリを入れておいたのでそれでタクシーを呼んでホテルまで移動。移動先の指定も支払いもアプリ内でできるので安心。

中心地に近づくにつれてバイクとクラクションの量が多くなる。さすがバイク天国。クラクションを「ちょっと通りますよ」のノリで鳴らしているので四方八方から聞こえる。なんなら運転手も普通に鳴らしている。

{{< figure src="/images/20260201/1/3.jpg" title="バイクと車の距離が近い" class="center" >}}

泊まるホテルはマジェスティックサイゴン。2025年で100周年を迎える由緒正しき5つ星ホテル。なんと1人1泊1万円ぐらいで泊まれた。ベトナムの物価の安さにビビる。

{{< figure src="/images/20260201/1/2.jpg" title="超綺麗" class="center" >}}

{{< figure src="/images/20260201/1/4.jpg" title="中も綺麗" class="center" >}}

荷物をおいて近場を散歩しつつ、早速フォーを食べに行った。

{{< figure src="/images/20260201/1/5.jpg" title="これが本場のフォー" class="center" >}}

一緒に香草が出てくるので、葉っぱをちぎってフォーに入れて食べる。沖縄そばにフーチバーを入れて食べるノリに似ているかも。

{{< figure src="/images/20260201/1/6.jpg" title="激うま" class="center" >}}

食べ終わってお会計する時に伝票も無いしレジも見当たらないのでどうすればいいか分からず…店員さんに「check please」と言っても伝わらずわちゃわちゃしてしまった。反省。

ホテルに戻って、ルーフトップのバーでミス・サイゴンを飲みました。創業当時からあるオリジナルカクテル。

{{< figure src="/images/20260201/1/7.jpg" title="ミス・サイゴン" class="center" >}}

{{< figure src="/images/20260201/1/8.jpg" title="バーからの景色。ホーチミンの夜景が綺麗" class="center" >}}

## 2日目

ホーチミン2日目。ベトナムと日本の時差は2時間で、時差ボケはあまり感じず普通に起きられた。

朝食はホテルのビュッフェ。サイゴン川を見ながら優雅に食べた。

ベトナムはあちこちにカフェがあるので歩き疲れたら気軽にカフェに入って休憩ができる。コーヒーも有名で、ベトナムコーヒーはとても濃厚で美味しい。

お昼はビンコムセンターというショッピングモールで良さげなお店に突入。

{{< figure src="/images/20260201/2/11.jpg" title="バインセオ。葉っぱで具材を巻いてタレにつけて食べる。" class="center" >}}
{{< figure src="/images/20260201/2/12.jpg" title="ブンチャー。麺をつけ汁につけて食べる。" class="center" >}}

色々ベトナム料理を食べたけど、ベトナム料理は楽しい！ちぎったり、巻いたり、何かにつけたり、手を動かして食べる事が多い気がする。

昨日の失敗を反省して事前にお会計の仕方をChatGPTに聞いておいた。どうやら店員さんに「Tính tiền！（ティン・ティエン）」と言えばよいらしいので実践。なんとか通じてお会計できた。嬉しい。

夜はベトナムのクラフトビールを飲みに行きました。夜は涼しくて、屋外の席で楽しく飲めました。

## 3日目
この日もホテルで朝食。すっかりベトナム料理にハマり、朝食からベトナム料理を食べる
{{< figure src="/images/20260201/3/1.jpg" title="" class="center" >}}

3日目もホーチミン観光。まずはサイゴン動植物園

{{< figure src="/images/20260201/3/2.jpg" title="シマウマ" class="center" >}}
{{< figure src="/images/20260201/3/3.jpg" title="すやすやのカバ" class="center" >}}
{{< figure src="/images/20260201/3/4.jpg" title="野良のベトナム猫" class="center" >}}

お昼は近くのレストランに移動。昨日食べたバインセオが忘れられず、もう一度食べる。美味い。

{{< figure src="/images/20260201/3/99.jpg" title="バインセオ" class="center" >}}

後で知ったけど、バインセオは南部（ホーチミン）だと大きくて中部（ダナン）は小さめらしい。このお店の本店はダナンなので、昨日食べたバインセオに比べて小さめだったっぽい。こういう違いも現地で実感できるの楽しいですね。

グエンチャイ通りという日本で言う原宿みたいな所でお買い物。自分は歩き疲れちゃったので途中で離脱してまたフォーを食べる。フォー食べすぎた。
{{< figure src="/images/20260201/3/7.jpg" title="食堂みたいな店のフォーが美味い" class="center" >}}

その後もまたぶらぶらしたりカフェで一休みしたり
{{< figure src="/images/20260201/3/6.jpg" title="ベトナムの町並み" class="center" >}}
{{< figure src="/images/20260201/3/8.jpg" title="フィンというコーヒーを淹れる器具" class="center" >}}

フィンで濃いめにゆっくり抽出して、たっぷりのコンデンスミルクを入れて飲むのがベトナムコーヒー。美味しい。

夕方はホーチミンの水上人形劇を鑑賞。面白かった。
{{< figure src="/images/20260201/3/9.jpg" title="水上人形劇場。観光客がたくさん。" class="center" >}}

夜はホテルに戻って近くの食堂でまたフォーを食べる。流石に食べすぎてこの頃には食傷気味になってた。
{{< figure src="/images/20260201/3/10.jpg" title="チェーン店のフォー" class="center" >}}

帰りにセブンイレブン視察。フルーツティーがたくさんあって、レジで氷を入れてくれる。異国のコンビニは現地の文化が見れて楽しいですね。
{{< figure src="/images/20260201/3/11.jpg" title="種類が豊富。日本にも欲しい" class="center" >}}

次の日からはベトナム国内線でダナンに移動。続きは別の記事で。
